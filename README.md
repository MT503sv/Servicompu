# sservicompu — Starter Next.js + Tailwind project

## Cómo usar

1. Descomprime este archivo.
2. Abre la carpeta en una terminal.
3. Ejecuta:
   ```bash
   npm install
   ```
4. Luego:
   ```bash
   npm run dev
   ```
5. Abre http://localhost:3000

## Notas
- Las imágenes de ejemplo están en `public/images/`. Añade tus imágenes ahí (product1.jpg, product2.jpg, product3.jpg).
- Si `npx tailwindcss init -p` falla en tu máquina, ya incluí `tailwind.config.js` y `postcss.config.js`.
- Si quieres TypeScript estricto u opciones adicionales, dímelo y lo ajusto.
